# 专题一：基于 FAST-LIO2 的室内环境建图

本项目是寒假机器人感知组的学习成果，成功在仿真环境下复现了激光 SLAM 建图流程。

## 🛠 实验环境
- **操作系统**: Ubuntu (Docker 容器环境)
- **中间件**: ROS Noetic
- **仿真器**: Gazebo (货架仓库场景)
- **传感器**: Livox Mid360 雷达 (仿真)

## 📈 技术细节
- **建图算法**: FAST-LIO2
- **点云处理**: 成功抓取并处理了约 750 万个激光点云数据。
- **地图转换**: 将 3D 点云 (`.pcd`) 投影转换为 2D 占据栅格地图 (`.pgm`)，分辨率为 0.05 m/pix。
